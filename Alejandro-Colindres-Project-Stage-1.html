HTML

HyperText Markup Language (HTML) is the standard language used to create web pages. Most of the web is based on this language.

HTML is made up of:

  1. Text content
  2. Markup(for edition)
  3. References to other documents(images or videos)
  4. Links

HTML Vocabulary

HTML documents are made of HTML elements. The types of elements are characterized by using HTML Tags.

  • Tags:They are used for holding content. An HTML tag is always contained within angled brackets. Most tags have an opening tag <> and a closing one </>, but some just have an opening one, this are called void tags.
  • Element: An HTML element refers to everything within a set of opening and closing tags.
  • Attributes: These are the properties of an HTML element.

Lesson 3: CSS Style and HTML Structure

CSS

Cascading Style Sheets (CSS) is used for setting the looks and format of a HTML document.

With CSS you can separte the document content from the document presentation. This helps avoiding repetition since it can easily apply similar elements with similar format.This is done by giving similar HTML elements the same class name and then specifying the style that should apply to that class.

Avoid repetition

Remember to avoid repetition. By avoiding it you avoid Copying and Pasting style attributes to elements which can lead to errors.

You also achieve consistency by having similar HTML elements show the same style.

Don't Forget!

Remember to link your HTML to a separate CSS file

Do this by:

  1. Write all your HTML in one file.
  2. Write all your CSS in another file.
  3. Add a <head> element to the top of the HTML document
  4. Add a <link> tag inside the <head> tag, containing the attributes rel="stylesheet" type="text/css" href="YOURFILENAME.css"
